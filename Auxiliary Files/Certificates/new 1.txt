Тестовый сертификат 9 марта 2021 создан для домена iit-fibertest
В Trusted Root Certification Authorities добавлен DATA CENTER IIT сертификат

в hosts добавлена строка
127.0.0.1       iit-fibertest  // при попытках обращаться к https по ip будет предупреждение

Запускаем WebApi
https://iit-fibertest:5001/WeatherForecast

При установке вэб сервера install пишет в /ini/settings.json
	{"apiProtocol":"http","apiPort":11080,"version":"2.1.0.814"}

При установке вэб сервера и выборе https режима

1. Запрашиваем пользователя где лежит pfx файл ключа - 
		копируем его в каталог WebApi\ini
		
2. Запрашиваем пароль к ключу, шифруем его и помещаем файл
	с зашифрованным паролем в каталог WebApi\ini
	
	при повторной установке проверять наличие pfx файла 
		и пароля к нему по этому пути
		
При запуске WebApi service использует эти файл и пароль для инициализации https слушателя
