SNMP agent - устройство поддерживающее протокол snmp, адресуется по IP и порту 
	(обычно на 161й получает команды/запросы и на 162й рассылает трапы)

snmpget - запросить у устройства значение хранящееся в ячейке с OID
	можно запросить значения сразу нескольких OID, возвращает лист значений	
	
snmpset - устанавливает на устройстве значение для ячейки с OID,
	можно послать значения сразу нескольких OID
	
snmpgetnext - ???

snmpgetbulk - используется вместо snmpgetnext начиная с v2

snmpwalk - состовная операция
		использует несколько snmpgetnext (v1) или snmpgetbulk (v2)
		получает список значений для поддерева OID
		
		
SNMP trap - обычно устройство (агент) рассылает трапы
SNMP listener - некий слушатель трапов, адресуется по IP и порту (обычно 162)		

snmpsendtrap - посылает trap (список значений для определенных OID) некоему слушателю

snmptrapd - слушатель трапов. оформить как службу и ок

snmpd is a SNMP agent which binds to a port and awaits requests from SNMP management software
